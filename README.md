# d.conomy – A Regenerative Economic Layer for the Symbiotic Commonwealth

**d.conomy** provides the economic backbone for the [Solarpunk Mandala](https://github.com/ravaioli/solarpunk-mandala) and [Symbiotic Commonwealth](https://solarpunksangha.com/academy/meta-narratives/sustainable-future). It replaces extractive private property (Game A) with decentralized, mutual‑aid‑based stewardship of resources (Game B).

- **Stewardship, Not Ownership** – Resources are held in trust for community benefit.
- **Decentralized, Not Nationalized** – Control resides in local assemblies (Guilds) and meshes upward via consent.
- **Visible Flow** – All economic activity is legible via the Φ‑Grid, making resource circulation as transparent as civic discourse.

## Core Components

| Component | Description |
|-----------|-------------|
| **Library of Things** | Community‑owned inventory of durable goods, borrowed rather than bought. |
| **Mutual Credit Circles** | Debt‑free exchange systems where members extend credit to one another. |
| **Community Land Trusts** | Nonprofits that hold land in trust for affordable, permanent community use. |
| **Φ‑Currency** | Complementary currency with demurrage, minted via Proof‑of‑Regeneration. |
| **Proof‑of‑Regeneration** | Protocol that issues new currency when measurable regenerative outcomes occur. |

## Relationship to CivicNotes & Mandala

- **CivicNotes** provides governance (Guilds, Promise Tracker) that manages d.conomy components.
- **Mandala** provides the ethical axes (Coherence, Care, Complexity, Courage) and the Tesseract geometry that ensures dimensional literacy.
