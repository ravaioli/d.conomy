# Proof‑of‑Regeneration (PoR)

## Description

A protocol that **issues new Φ‑currency** when a measurable regenerative outcome is achieved. PoR turns visible regeneration into a tangible economic reward, creating a positive feedback loop.

## Types of PoR Events

| Event Type | Verification Method | Minting Formula |
|------------|---------------------|-----------------|
| **Community Harvest** | Guild‑verified yield report | 1 Φ per lb of food grown |
| **Promise Addressed** | Promise Tracker verification | 100 Φ per promise |
| **LoT Usage Milestone** | Automatic from LoT system | 1 Φ per 50 borrows |
| **New CLT Acquisition** | CivicNotes petition approval | 1000 Φ per acre |
| **Mutual Credit Circle Formation** | Guild Council recognition | 500 Φ per circle |

## Verification

- Most events are verified through the **Promise Tracker** community‑verification mechanism (see CivicNotes governance).
- For automated events (e.g., LoT usage), the system records milestones without manual verification.
- Disputes are handled by the Moderation Council.

## Economic Impact

- PoR minting expands the money supply only in response to regenerative activity, ensuring value is tied to life, not debt.
- It provides a funding source for Guilds and individuals who contribute to regeneration.
- It makes the “invisible” work of stewardship economically visible.

## Technical Implementation

- **Event detection** via CivicNotes APIs (Promise Tracker) and LoT system.
- **Minting smart contract** that issues Φ‑currency to designated recipients (e.g., the Guild that completed the action).
- **Audit log** of all PoR events, viewable on the Φ‑Grid.
