# Φ‑Currency

## Description

A **complementary currency** designed to circulate, not accumulate. Demurrage (a small holding fee, e.g., 0.5% per month) incentivizes spending and investment in regenerative activities.

## Minting

New Φ‑currency is minted exclusively through **Proof‑of‑Regeneration (PoR)** events:

- A community garden yields 500 lbs of food → 500 Φ minted.
- A Promise Tracker item is verified “addressed” → 100 Φ minted.
- A Library of Things tool is borrowed 50 times → 50 Φ minted.

Minting parameters (e.g., exchange rates, maximum supply) are set by the Guild Council and reviewed via CivicNotes.

## Demurrage

- A holding fee is deducted from all balances periodically (e.g., monthly).
- Demurrage rates adjust based on circulation velocity: too slow → increase demurrage to encourage spending.
- Collected demurrage is redistributed to active users or used to fund community projects (subject to CivicNotes vote).

## Governance

- The **Guild Council** sets minting rules and demurrage rates.
- **CivicNotes** provides the forum for policy changes and dispute resolution.
- **Promise Tracker** verifies PoR events, which trigger minting.

## Technical Implementation

- **Digital ledger** (blockchain or centralized) tracking balances and demurrage.
- **API** for transfers, balance queries, and PoR event registration.
- **Integration** with CivicNotes to display individual and Guild balances on user profiles.
