# Overview

## Vision

d.conomy creates a **visible, accountable, regenerative economy** where communities own and manage resources collectively. It replaces the logic of extraction and scarcity with stewardship and flow, making economic activity a visible part of community health (the Φ‑Grid).

## Core Principles

1. **Stewardship, Not Ownership**  
   Resources are held in trust by communities. Private property is transformed into commons through legal structures like Community Land Trusts and shared inventories like Libraries of Things.

2. **Decentralized, Not Nationalized**  
   Control is local. Guilds (via CivicNotes) manage resources, and these local nodes mesh into bioregional confederations without central hierarchy.

3. **Mutual Aid, Not Debt**  
   Exchange happens through mutual credit and Φ‑currency with demurrage, avoiding interest‑based debt and encouraging circulation.

4. **Visible Flow**  
   All economic activity feeds the Φ‑Grid, making resource flows as transparent as civic discourse. This enables real‑time feedback and accountability.

## How It Works with CivicNotes

CivicNotes provides the **governance interface**:

- **Guilds** manage d.conomy components (e.g., a Library of Things Guild).
- **Promise Tracker** verifies Proof‑of‑Regeneration events (e.g., a community garden reaching harvest).
- **Petitions** can propose new d.conomy initiatives (e.g., create a mutual credit circle).

In return, d.conomy provides the **material infrastructure** that makes CivicNotes promises actionable.

## Phased Transition

See Appendix Z of the Mandala for the full dual‑power strategy. d.conomy follows the same phases:

- **Phase 1 (Cultural Prefiguration)** – Pilot Libraries of Things, small mutual credit circles.
- **Phase 2 (Functional Substitution)** – Expand CLTs, introduce Φ‑currency for local exchange.
- **Phase 3 (Systemic Integration)** – Mesh into bioregional commons; Φ‑currency becomes default local medium.
